# **App Name**: MDMAK Home

## Core Features:

- Dynamic Landing Page Display: Render all specified sections (Hero, Categories Grid, How It Works, etc.) with a responsive, mobile-first layout using Next.js, TailwindCSS, and ShadCN UI components.
- Multilingual & RTL Support: Implement client-side language switching, making Arabic the primary (RTL) language and English the secondary (LTR), affecting all textual content and layout direction.
- Interactive Marketplace Showcases: Display dynamic grids for 'Categories', 'Featured Products', and 'Contractor Marketplace' sections with sample data and clickable elements to demonstrate browsing capabilities.
- AI-powered Material Cost Estimator Tool: An interactive, generative AI-driven tool allowing homeowners to input basic project parameters (e.g., area, material type) and receive estimated cost ranges or material quantity recommendations.
- Direct Quotation Request Forms: Enable homeowners to submit simplified forms directly from the landing page to request quotations for materials or services, capturing essential contact information.
- Comprehensive Call-to-Action Integration: Embed prominent and engaging CTAs throughout the page to guide users to key actions such as 'Browse Products', 'Request Quotes', or exploring contractor listings.
- Dynamic Animation with Framer Motion: Incorporate subtle, premium-feeling animations for key UI elements and section transitions using Framer Motion to enhance the user experience without distraction.

## Style Guidelines:

- Primary color: A deep, sophisticated blue (Hex #263A6E) reflecting professionalism and trust, inspired by premium SaaS brands and aligning with MDMAK Home's core navy.
- Background color: An almost-white background (Hex #F5F7FA) with a subtle cool undertone, maintaining a clean and minimal aesthetic while softly connecting to the primary blue.
- Accent color: A vibrant and fresh cyan (Hex #30DBF9) provides energetic contrast and highlights interactive elements, conceptually similar to MDMAK Home's brand cyan. The brand's success green (Hex #10B981) will be reserved for confirmation messages and positive feedback.
- Body and headline font: 'Inter', a grotesque-style sans-serif for its modern, machined, objective, and neutral aesthetic, suitable for both headlines and body text to maintain a premium and minimal feel.
- Use clean, consistent vector icons (line-art or subtly filled) that align with the modern and minimal design language, ensuring clarity across different screen sizes, consistent with leading SaaS platforms.
- An adaptive, mobile-first layout prioritizing content readability and interaction on smaller screens, seamlessly transitioning to desktop. The layout will natively support Arabic RTL content flow, employing generous spacing and large rounded corners for all UI components. Soft shadows will provide subtle depth and a premium touch.
- Subtle and purposeful animations using Framer Motion will be implemented for smooth section transitions, hover effects, and feedback on interactive elements, enhancing the premium feel without being distracting or impeding usability.
